# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for HeadlessCraft.

## ADR Lifecycle

- **Proposed** → Under discussion
- **Accepted** → Ratified and in effect
- **Superseded** → Replaced by a newer ADR (never edited — create a new one)
